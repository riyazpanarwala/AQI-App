# AQI India Live – Real-Time Air Quality Map App

A beautiful, fast, and completely free app that shows **real-time Air Quality Index (AQI)** at your current location and nearby monitoring stations across India — specially designed for rural areas affected by crop burning, traffic, and dust.

### Features
- Real-time AQI powered by **waqi.info** (best rural station coverage in India)
- Interactive full-screen map with color-coded AQI markers (Green → Red → Maroon)
- Shows nearest station even in remote villages
- Your live location with blue dot
- Refresh & WhatsApp share button
- Works as **Web App + Android + iOS** from single code
- PWA – installable on any phone (even JioPhone & low-end devices)
- No login, no ads, no tracking
